# Student Accommodation Analytics Dashboard

## Objective
Analyse a 10,000+ record student accommodation dataset to identify inefficiencies in offer processing and allocation.

## Tools
- Power BI  
- Power Query (ETL)

## What I Did
- Cleaned and transformed raw data using Power Query (dates, capacity, applicant type)  
- Built interactive dashboards to track offer flows, contracts, and terminations  
- Analysed trends across applicant groups and time periods  

## Key Insight
- Priority applicant groups (Indigenous and scholarship students) experienced the longest delays, revealing a gap between institutional intent and operational outcomes  
- Suggests need for improved prioritisation logic and process optimisation in allocation workflows
